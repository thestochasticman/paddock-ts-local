# Phenology

Per-paddock phenology metrics — start/end of season, peak position,
amplitude, length of integral over season, etc. — computed via
[phenolopy](https://github.com/lewistrotter/phenolopy) by Lewis Trotter
(Apache 2.0-licensed; see `PaddockTS/LICENSES/phenolopy.LICENSE`),
vendored under `PaddockTS/Phenology/_phenolopy.py`.
