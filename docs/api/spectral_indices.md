# Spectral indices

Five vegetation/soil indices computed from Sentinel-2 surface reflectance:

- **NDVI** — Normalised Difference Vegetation Index
- **CFI** — Crop Foliage Index
- **NIRv** — Near-Infrared Reflectance of Vegetation
- **NDTI** — Normalised Difference Tillage Index
- **CAI** — Cellulose Absorption Index

All indices are computed in-memory and added as new variables to the
Sentinel-2 dataset.
